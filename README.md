# CERNO Website - LAB Case Club

A modern, responsive website for the LAB Case Club at CERNO European Business Club.

## Project Structure

```
CERNO Website/
├── index.html              # Main HTML file
├── css/
│   └── styles.css         # Main stylesheet
├── js/
│   └── main.js            # JavaScript functionality
├── images/
│   └── business-meeting.jpg  # Images and photos
├── assets/
│   ├── fonts/             # Custom fonts (if needed)
│   └── icons/             # Icon files (SVG, PNG, etc.)
└── README.md              # This file
```

## Features

- Responsive design that works on desktop, tablet, and mobile
- Modern navigation with sticky header
- Hero section with call-to-action
- About section with image gallery
- Footer with contact information and social links

## How to Use

1. Open `index.html` in your web browser
2. Replace placeholder image in `images/` folder with actual content
3. Customize content, colors, and styles in `css/styles.css`
4. Add custom JavaScript functionality in `js/main.js`

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
